# Investment Calculator

A Streamlit dashboard for planning and visualizing investment growth.

## Features

- **Compound Interest** — Calculate growth from a lump-sum investment with configurable compounding frequency
- **SIP / Regular Contributions** — Model portfolio growth with recurring monthly contributions
- **Goal Planner** — Work backwards from a target amount to find the required monthly contribution

## Getting Started

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tech Stack

- [Streamlit](https://streamlit.io/) — dashboard framework
- [Plotly](https://plotly.com/python/) — interactive charts
- [Pandas](https://pandas.pydata.org/) / [NumPy](https://numpy.org/) — data handling
